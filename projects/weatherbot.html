WeatherBet — Polymarket Weather Trading Bot

Automated weather-market trading engine designed for signal quality, strict risk controls, and operational reliability. Built in Python with market data integration, forecast fusion, and deployment-ready runtime packaging.

Overview

WeatherBet scans weather prediction markets, maps forecast signals to exact temperature buckets, and executes strategy decisions using expected value and position-sizing controls. The architecture supports both paper-mode operation and guarded live-mode behavior.

Core capabilities

  • Forecast fusion across multiple sources with airport-specific market alignment.
  • Expected value filtering and Kelly-based bet sizing with max-bet constraints.
  • Stop-loss, trailing stop, and forecast-change exit logic.
  • Persistent JSON state and trade records for auditability and post-run analysis.
  • Geoblock-aware live-mode safeguards to prevent repeated failed order attempts.
  • Dockerized deployment using Dockerfile + Compose for reproducible migration.

Tech stack

Python, requests, py-clob-client, web3, dotenv, Docker, Docker Compose.
